## Qot_GetBroker

### 示例

```go
futugg.Cmd("send.Qot_Sub", cli, "HK.01810", "Broker", true, true, "None", false)
futugg.Cmd("send.Qot_GetBroker", cli, "HK.01810")
```

### Request参数

参数名  | 类型  | 是否必须 | 说明      | 示例
------- | ---- | -------- | -------  | ---------
stockCode | string | 是 | 股票名称，固定格式 | HK.01810
